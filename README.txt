BejeweledByJoy App (Full Version)
Features:
- Supabase Auth (Owner + multiple admins)
- Admin Dashboard (product management, analytics)
- Product CRUD with image uploads to Supabase Storage
- Mobile-first layout
- Public product view with add-to-cart (no login required)
- WhatsApp order link
- Gallery with multiple images per product
- Discount badges with styled CSS
- SQL schema to set up Supabase tables

Setup instructions provided in the README_SETUP.md file.
